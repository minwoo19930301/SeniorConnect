# OpenAI Build Week Submission Checklist

Source of truth: [OpenAI Build Week rules](https://openai.devpost.com/rules). Recheck the live page immediately before submitting.

## Event fit

- [ ] Proposed track: **Apps for Your Life**.
- [ ] The submission describes one coherent product, not four unrelated demos.
- [ ] Codex materially contributes to the build workflow.
- [ ] GPT-5.6 materially powers Speak, Camera, or constrained intent proposals.
- [ ] The product is functional and testable by judges.
- [ ] One team representative submits for the four-person team.

## Repository

- [ ] Project and code name is `an-agent-for-elders`; the repository name may remain unchanged.
- [ ] README explains setup, architecture, safety boundaries, and how to test.
- [ ] MIT license remains present.
- [ ] No credentials, real contacts, private images, or personal messages are committed.
- [ ] `npm test` and future application checks pass.
- [ ] Public repository is ready, or the required judging accounts have access if private.

## Submission materials

- [ ] English project description.
- [ ] Public YouTube demo is no longer than three minutes.
- [ ] Demo has narration or other audio.
- [ ] Demo shows the working product, Codex contribution, and GPT-5.6 usage.
- [ ] Demo uses only synthetic personal data and permitted media.
- [ ] Working free demo or testable build is available to judges.
- [ ] Primary Codex task includes `/feedback`; its session ID is included.
- [ ] One track is selected.

## Deadline

Submit before **July 21, 2026 at 5:00 PM PDT**, which is **July 22, 2026 at 9:00 AM KST**. Leave time for video processing, permission checks, and a clean-device test.

## Final rehearsal

- [ ] Install or open from the exact judge instructions.
- [ ] Run Call, YouTube, Speak, and Camera once each.
- [ ] Demonstrate one denial or uncertainty case.
- [ ] Verify source links and network failure behavior.
- [ ] Confirm no planned feature is presented as implemented.
- [ ] Submit, then open the public entry and every link in a signed-out browser.
