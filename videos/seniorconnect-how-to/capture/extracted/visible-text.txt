Current repository facts that the walkthrough must preserve:

- The project is a plan for a simple Android assistant for older adults.
- There is no working Android app or server yet.
- The planned home screen has four large choices: Talk, See, Family, and Help.
- Talk supports simple conversation and source-backed web answers.
- See explains user-requested photos of text, signs, menus, labels, and controls.
- Family opens a preselected trusted contact after clear confirmation.
- Help explains confusing screens, advertisements, and possible install buttons.
- The user stays in control. The agent does not silently call, message, install,
  purchase, block advertisements, or press buttons in other apps.
